# SafeALU

## Aim

To design an 8-bit Arithmetic Logic Unit (ALU) with overflow detection.

## Description

- Performs arithmetic and logical operations.
- Supports addition, subtraction, AND, OR, XOR, NOT, left shift, and right shift.
- Generates an overflow flag during addition.

## Inputs

- A[7:0]
- B[7:0]
- sel[2:0]

## Outputs

- result[7:0]
- overflow

## Test Cases

- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift

## Result

The SafeALU successfully performs arithmetic and logic operations and indicates overflow during addition.
